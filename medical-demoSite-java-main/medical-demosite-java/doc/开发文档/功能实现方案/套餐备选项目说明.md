制定套餐，项目勾选备选后，必须录入备选分组、组类型、优惠价。  
一个备选分组只能有一种组类型  
#组类型  
##组间选  
登记时，
组间选项目必须按组选择，选择一个分组中的一个项目时，必须选择这个分组所有项目。
选择的总组数不能超过【备选组数量】。  

创建套餐时，
必须录入备选组价格、备选组数量，备选组数量必须大于0。   
【备选组数量】不能大于【组间选】类型的分组数。  
每个组间选分组的所有收费项目优惠价总和必须等于备选组价格。  

##组内选  
登记时，
【组内选】类型每个备选分组最多选择一个项目。   

创建套餐时，  
同一个分组内所有收费项目的优惠价必须相同。  

##组任选   
登记时，
可以随便选。   

创建套餐时，
可以随便录入。    


#套餐原始价格计算   
所有非备选项目原价+（组内1组原价合计/组内1组项目数+组内2组原价合计/组内2组项目数+...）+（所有组外选项目原价*可选数量/所有组外选组数）
