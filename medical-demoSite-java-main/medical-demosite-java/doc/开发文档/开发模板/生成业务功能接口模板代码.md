## 生成业务功能接口模板代码

- 直接放在功能主Controller页面中
- 示例

```java
/**
 * 总检管理-健康总检功能接口总结
 */
@GetMapping("/getInterfaceList")
@ApiOperation(value = "接口说明", notes = "获取【总检管理-健康总检】这个块业务所有接口及接口说明")
public R<FunctionVo> getInterfaceList(){
        List<InterfaceVo> interfaceVos=Arrays.asList(
        new InterfaceVo("分页获取待总检人员列表数据","GET","/total/health/page","10.总检-报告系统->总检管理-健康总检->分页查询",null),
        new InterfaceVo("获取总检结论词列表数据","GET","/total/verdict/list","10.总检-报告系统->总检结论词->获取结论词列表",null));

        return R.ok(new FunctionVo("10.总检-报告系统","总检管理",interfaceVos,"10.总检/报告系统"));
        }
```
