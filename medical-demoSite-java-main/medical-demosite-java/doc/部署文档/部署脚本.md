# java服务项目启动命令

- 线下主系统服务启动命令

```shell
# 正式环境
# 8080
# mcweb-admin
# 新体检系统接口主服务(正式)
/usr/local/btjdk/jdk8/bin/java -jar -Xmx2048M -Xms512M  -Dspring.profiles.active=common,mq,prod /www/wwwroot/java_8080/mcweb-admin.jar --server.port=8080

# 测试环境
# 8090
# mdt-admin
# 新体检系统接口主服务(测试)
/usr/local/btjdk/jdk8/bin/java -jar -Xmx1024M -Xms256M  -Dspring.profiles.active=common,mq,dev /www/wwwroot/java_8090/mdt-admin.jar --server.port=8090
```

- 线下独有系统服务启动命令

/usr/local/btjdk/jdk8/bin/java -jar -Xmx1024M -Xms256M -Dspring.profiles.active=dongqu /www/wwwroot/java_8082/mcweb-center.jar --server.port=8082

- 线下DICOM服务启动命令

```shell
# mcweb-dicom-admin
# DICOM图片接收服务(正式)

# 后台启动
@echo off
start javaw -Dfile.encoding=UTF-8 -jar mcweb-dicom-admin.jar --spring.profiles.active=prod
exit

# 控制台启动
java -jar -Dspring.profiles.active=dr mcweb-dicom-admin.jar
```
